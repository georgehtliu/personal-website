# georgehtliu.xyz

My personal website - a space-themed, interactive portfolio built with vanilla HTML, CSS, and JavaScript.

## Features

- **Space aesthetic** - Animated stars, galaxy spiral, gravity waves, and warp-speed lines
- **Interactive dog** - A space dog companion you can feed, play with, unleash, and give zoomies to
- **Typewriter hero** - Animated name intro on load
- **Experience timeline** - Scrollable career journey with company logos
- **Effects toggle** - Turn visual effects on/off

## Tech Stack

- HTML, CSS, JavaScript (no frameworks)
- Hosted via GitHub Pages with a custom domain
